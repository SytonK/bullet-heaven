experience:
killing enemies grant experience. earning enough experience levels up the player. when the player levels up he can choose a new [ability](obsidian://open?vault=bullet-heaven-gdd&file=abilities) / [upgrade](obsidian://open?vault=bullet-heaven-gdd&file=upgrades)

mine:
throughout the run there will be stones to mine. some of those stones can hold experience or other run bonuses
